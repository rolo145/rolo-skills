# dep-guard-update

Skill for updating npm dependencies via the [dep-guard](https://github.com/rolo145/dep-guard) CLI.

## Prerequisites

Install dep-guard:

```bash
npm install -g @roland.botka/dep-guard
```

- GitHub: https://github.com/rolo145/dep-guard
- npm: https://www.npmjs.com/package/@roland.botka/dep-guard
